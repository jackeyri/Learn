## 多用户
1. 同时可以有多个用户来登陆操作系统
2. 用户的类型 
3. 超级用户 root 
   - 类似于Windows 下的 Administrator
   - 拥有最高的权限，格式化磁盘，安装软件，关闭重启系统，创建删除其他用户
   - 可以操作所有的目录
   - 家目录 /root
   - UID 0
4. 普通用户
   - 没有上述的权限
   - 只能操作自己的家目录
   - 家目录 /home/用户名
   - sudo 临时把普通用户的权限提升到root权限 / 输入普通用户的密码
   - 不是所有的普通用户都可以使用sudo
   - UID
5. 虚拟用户
   - UID，家目录
   - 不能登录系统，无法操作系统
